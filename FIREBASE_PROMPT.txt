DRAG & DROP CODE:
```javascript
class TaskBoardApp {
  handleDragStart(e) {
    this.draggedCard = e.target;
    e.dataTransfer.effectAllowed = 'move';
  }
  handleDrop(e) {
    const listId = e.target.closest('.list').dataset.listId;
    this.moveCard(this.draggedCard.id, listId);
  }
}
```
